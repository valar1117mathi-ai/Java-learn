# Java + DSA — Learning Hub

A structured reference for learning Data Structures and Algorithms using Java.
Designed for programmers who know basic Java and want to build a solid DSA foundation.

---

## What is DSA?

**Data Structures** are ways to organize and store data so it can be accessed and modified efficiently.

**Algorithms** are step-by-step procedures to solve a problem or perform a computation.

Together, DSA determines **how fast** and **how efficiently** a program runs — critical for technical interviews, competitive programming, and real-world systems.

---

## Why Java for DSA?

- **Strongly typed** — forces clarity about data types and reduces ambiguous bugs.
- **Rich standard library** — `java.util` provides production-grade implementations of lists, maps, queues, and more.
- **Object-oriented** — natural fit for modeling nodes, graphs, and trees.
- **Widely used in interviews** — accepted on LeetCode, HackerRank, Codeforces, and most coding rounds.
- **Garbage collection** — no manual memory management, so you focus on logic.

## Complexity Quick Reference

| Notation | Name | Example |
|----------|------|---------|
| O(1) | Constant | Array index access |
| O(log n) | Logarithmic | Binary search |
| O(n) | Linear | Linear scan |
| O(n log n) | Linearithmic | Merge sort |
| O(n²) | Quadratic | Bubble sort |
| O(2ⁿ) | Exponential | Brute-force recursion |
